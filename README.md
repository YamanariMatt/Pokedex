## 🧠 Sobre o Projeto

Esta aplicação é uma Pokédex interativa que permite explorar Pokémons, visualizar detalhes completos, filtrar por tipo e salvar favoritos.

O projeto foi desenvolvido com foco em:
- Consumo de API REST
- Interface responsiva (mobile-first)
- Experiência de usuário fluida
- Design retrô inspirado nos jogos clássicos

---

## ⚙️ Tecnologias Utilizadas

- HTML5
- CSS3 (estilização customizada retrô)
- JavaScript (ES6+)
- React (via CDN)
- Babel (transpilação no navegador)
- API: https://pokeapi.co/

---

## 🎯 Funcionalidades

### 🔍 Busca
- Busca por nome ou ID do Pokémon
- Tratamento de erro para Pokémon inexistente

### 📋 Listagem
- Exibição inicial de Pokémons
- Paginação com botão "Carregar mais"

### 🧬 Detalhes
- Modal com informações completas:
  - Nome, ID e imagem
  - Tipos
  - Altura e peso
  - Habilidades
  - Estatísticas com barras visuais
  - Movimentos
  - Cadeia de evolução

### ⭐ Favoritos
- Marcar/desmarcar Pokémon como favorito
- Persistência com LocalStorage
- Visualização exclusiva de favoritos

### 🎨 Interface
- Design retrô estilo Pokédex
- Fonte pixelada (Press Start 2P)
- Animações suaves
- Tema claro/escuro

### 📱 Responsividade
- Totalmente adaptado para:
  - Celulares
  - Tablets
  - Desktop
